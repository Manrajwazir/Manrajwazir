### 👋 About

Engineer who likes hard problems and shipping real things — not just writing code, but designing the system around it. I work across the stack (backend, APIs, data, frontend, cloud), think in architecture and trade-offs, and reach for AI — agentic tooling or computer vision — when the problem actually calls for it. Currently building a company while finishing my CS diploma.
